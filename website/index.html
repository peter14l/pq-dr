CRYPTO STACK

Defense In Depth Architecture

We combine proven classical cryptography with next-generation post-quantum standards. If one algorithm fails, the other holds the line.

🌀

ML-KEM-1024

FIPS 203 standardized post-quantum Key Encapsulation Mechanism. Represents the highest tier of quantum-resistant security.

🔑

X25519 Fallback

Classical Diffie-Hellman backup. Ensures absolute security in the highly unlikely event of a mathematical exploit in PQC.

🔒

AES-256-GCM-SIV

Symmetric cipher resistant to nonce-misuse. Prevents accidental leakage even if developers reuse nonces during mass data transmissions.

🧼

Zeroize Memory Wiping

Automatic memory wiping of cryptographic private keys and symmetric key materials as soon as they are dropped, preventing RAM reading attacks.

🕶️

Header Encryption

Unlike standard Signal protocol implementations, PQ-Aura encrypts message headers, completely hiding the ratchet state and packet metadata from intermediary servers.

💾

Atomic Persistence

Secure session serialization and deserialization with "Shadow Saves" prevents file corruption and database leaks on persistent disks.

DEVELOPER REFERENCE

Built for Mobile & Web Developers

Drop our native FFI packages directly into your codebase. Here is how easy it is to initialize a secure session. 

import 'package:pq_aura_flutter/pq_aura_flutter.dart';

// 1. Generate local hybrid keys (X25519 + ML-KEM-1024)
PqKeyPair aliceKeyPair = PqAuraClient.generateKeyPair();

// 2. Fetch Bob's pre-key bundle from your server
PqPreKeyBundle bobBundle = PqPreKeyBundle.fromJson(bobBundleJson);

// 3. Initiate the handshake
var handshake = PqAuraClient.initAlice(
  remoteBundle: bobBundle,
  localIdentityKeyPair: aliceKeyPair,
);

PqSession aliceSession = handshake.session;
PqInitialMessage initialMsg = handshake.initialMessage;

// 4. Encrypt message payload
PqMessage encrypted = aliceSession.encrypt(
  Uint8List.fromList("Hello Bob!".codeUnits),
  associatedData,
);
use pq_aura::handshake::HandshakeEngine;
use pq_aura::RatchetEngine;

// Alice initiates the session using Bob's bundle
let (mut alice_state, initial_msg, root_key) = 
    HandshakeEngine::initiate_alice(
        &bob_bundle, 
        &alice_id_pk, 
        &alice_id_sk, 
        &mut rng
    );

// Encrypt message
let message = RatchetEngine::encrypt(
    &mut alice_state, 
    b"Hello Bob!", 
    associated_data, 
    &mut rng
);
#include <pq_aura.h>

// Generate keypair
FfiKeyPair* keys = pqa_generate_keypair();

// Encrypt message
FfiMessage* msg = pqa_encrypt(
    state, 
    plaintext, 
    plaintext_len, 
    ad, 
    ad_len
);

// Clean up memory
pqa_free_message(msg);
pqa_free_keypair(keys);
